# PrintBit End-User Flow Guide

## 1) Print Method

1. Open **Print** on the kiosk.
2. Scan the on-screen QR code with your phone and upload a file.
3. Wait for the file to appear in **Received files**.
4. Select one file, then tap **Continue to settings**.
5. In Config, choose print options (color mode, copies, orientation, paper size, page range).
6. Continue to Confirm, insert coins, and confirm when balance is enough.
7. Collect printed pages from the output tray.

Troubleshooting:
- No uploaded file appears: start a **New session** and upload again.
- Insufficient balance: insert more coins before confirming.
- If there are remaining files after one print, return to Print and repeat for the next file.

## 2) Copy Method

1. Open **Copy** on the kiosk.
2. Place document face-down on the scanner glass.
3. Tap **Check Document** to run preview scan.
4. Review preview on the left panel.
5. If preview is correct, tap **Continue to Config**.
6. Choose copy settings, proceed to Confirm, insert coins, then confirm.
7. Collect copied pages.

Troubleshooting:
- No document detected: reposition document and tap **Retry**.
- Preview is unclear/cropped: run **Check Document** again before continuing.
- Insufficient balance: insert more coins, then confirm again.

## 3) Scan Method

1. Open **Scan** on the kiosk.
2. Choose scan settings (source, color mode, resolution).
3. Place document, then tap **Scan Document**.
4. Review output preview; tap **Rescan** when needed.
5. Tap **Get Soft Copy**.
6. Choose delivery method:
   - **Wireless (QR):** scan QR code on phone to download.
   - **USB Flash Drive:** insert USB drive, refresh drive list, export scan.

Troubleshooting:
- Scanner unavailable: check scanner power/USB connection and retry.
- Wireless link expired: tap **Refresh QR Link**.
- USB not listed: insert/reinsert USB drive and tap **Refresh USB**.

## Shared Payment and Confirmation Notes

- Print and Copy require sufficient coin balance before confirmation.
- Confirm screen shows current balance and required total.
- If balance is lower than required amount, users must add more coins before confirming.
